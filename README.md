# 🌠 星光承诺 StarlightPromise


## 项目介绍

《星光承诺》是一款使用 Python + Kivy 开发的小型剧情探索 RPG 游戏。

目标：

制作一款可以运行在 Android 手机上的治愈系 RPG 游戏。


---

## 项目信息

StarlightPromise

1.0


引擎：

Python + Kivy


平台：

- Windows
- Android


类型：

剧情探索 RPG


---

## 核心玩法

玩家控制主角：

- 探索地图
- 寻找物品
- 触发剧情
- 解开回忆
- 收集星光


最终完成属于两个人的故事。
