# 🌌 Starlight Promise

# Story Design Document

版本：1.0


# 1. 故事标题

《星光之约》


# 2. 故事主题

这是一个关于：

- 回忆
- 陪伴
- 约定
- 珍惜

的短篇 RPG 故事。


# 3. 世界观

在一个被星光照耀的小镇中。

传说：

每一个重要的回忆都会变成一颗星光碎片。

收集所有碎片的人，可以打开“星光之门”。

星光之门的另一边，保存着最珍贵的祝福。


# 4. 主线剧情


## 序章：星光降临

玩家在夜晚醒来。

发现手中出现一颗神秘的星光。

一个声音告诉玩家：

“还有一个重要的约定，需要你完成。”


---

# 第一章：寻找失落的星光

地点：

星光村


目标：

寻找第一颗星光碎片。


玩家探索：

- 村庄
- 花园
- 小路


通过帮助居民，获得线索。


获得：

第一颗星光碎片。


---

# 第二章：回忆森林

地点：

回忆森林


目标：

寻找隐藏的记忆。


玩法：

- 解谜
- 探索
- 收集物品


玩家发现：

每一片星光，都记录着一段美好回忆。


---

# 第三章：星光湖

地点：

星光湖


目标：

完成最后的试炼。


玩家需要：

- 解开湖边谜题
- 找齐所有碎片


最终获得：

星光钥匙。


---

# 最终章节：星光之约


地点：

星光花园


玩家打开星光之门。


所有收集的回忆汇聚。


屏幕出现：

“谢谢你，一直陪伴我。”


随后播放最终动画。


显示：

# 生日快乐


游戏结束。
