# Quest Design Document

# 1. 任务系统概述


游戏任务分为：


- 主线任务
- 支线任务
- 探索任务
- 解谜任务
- 最终任务



任务目标：

通过完成任务推进剧情。

收集全部星光碎片。

开启最终生日祝福。



---

# 2. 主线任务流程


## Quest 001

# 消失的星光


章节：

序章


地点：

星光小屋


任务目标：

发现神秘星光。


流程：


1. 玩家醒来。

2. 调查桌上的信。

3. 与星灵 Luma 对话。

4. 开启寻找星光碎片任务。


奖励：

开启地图：

星光村



---

# Quest 002

# 星光村的秘密


章节：

第一章


地点：

星光村


任务目标：

寻找第一颗星光碎片。


流程：


1. 与诺亚老人对话。

2. 帮助村民。

3. 收集线索。

4. 找到隐藏地点。


奖励：

第一颗星光碎片。


解锁：

回忆森林。

# Quest 003

# 遗失的花种


类型：

支线任务


NPC：

艾琳


目标：

寻找森林中的特殊花种。


流程：

1. 前往森林。

2. 找到隐藏区域。

3. 带回花种。


奖励：

# Quest 004

# 回忆森林的谜题


章节：

第二章


地点：

目标：

流程：

1. 寻找三个记忆石。

2. 根据提示排列。

3. 打开隐藏道路。


奖励：

第二颗星光碎片。
