# 🌌 Starlight Promise

# UI Design Document

版本：1.0


# 1. UI设计目标


目标：

制作适合安卓手机操作的2D RPG界面。


设计原则：

- 简洁
- 温馨
- 易操作
- 符合剧情氛围



---

# 2. 屏幕适配


目标设备：

Android手机


默认分辨率：

1920 × 1080


支持：

- 横屏模式
- 不同尺寸屏幕


UI采用：

Kivy布局系统



---

# 3. 主菜单界面


进入游戏时显示。


包含按钮：


## 开始游戏


功能：

创建新游戏。


---

## 继续游戏


功能：

读取存档。



---

## 设置


功能：

调整：

- 音量
- 文字速度
- 操作方式



---

## 关于


显示：

游戏名称

版本

制作信息

- NPC
