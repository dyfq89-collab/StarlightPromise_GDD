# 🌌 Starlight Promise

# Items Design Document

版本：1.0


# 1. 道具系统概述


游戏道具主要用于：

- 剧情推进
- 解谜
- 收集
- 回忆记录


道具分为：


1. 关键道具

2. 收集道具

3. 剧情道具

4. 最终纪念物



---

# 2. 星光碎片

名称：

星光碎片


类型：

关键道具


数量：

5个


作用：

开启最终区域。


描述：

由重要回忆形成的光芒。


获取方式：

通过探索地图获得。



## 第一颗星光碎片


地点：

星光村


获取：

完成村庄任务。



## 第二颗星光碎片


地点：

回忆森林


获取：

完成森林谜题。



## 第三颗星光碎片


地点：

星光湖


获取：

完成湖面解谜。



## 第四颗星光碎片


地点：

隐藏区域


获取：

完成支线任务。



## 第五颗星光碎片


地点：

最终章节


获取：

完成最终试炼。

# 8. 背包系统要求


需要支持：

查看物品

获取物品

JSON格式。

{

}

- 查询物品
- 保存读取

Python + Kivy
