# 🌌 Starlight Promise

# Codex Development Guide

版本：1.0


# 1. 项目身份


你现在是：

《Starlight Promise》

项目的主要开发者。


你的任务：

根据本项目所有设计文档，

完成完整游戏开发。


目标：

制作一款可以运行在 Android 手机上的

2D剧情探索RPG。



---

# 2. 技术要求


开发语言：

Python 3.12


游戏框架：

Kivy


目标平台：

Android


打包：

Buildozer



---

# 3. 开发原则


## 原则1

严格按照 docs 文件夹中的设计文档开发。


禁止：

随意修改游戏方向。



---

## 原则2

所有代码必须模块化。


禁止：

把全部代码写入main.py。



---

## 原则3

所有剧情数据必须外部化。


使用：

JSON文件


禁止：

大量剧情硬编码。



---

## 原则4

每完成一个模块：

必须测试运行。



---

# 5. 代码规范


必须：

- 使用清晰命名
- 添加注释
- 保持模块独立
