# 2. 游戏核心目标

玩家将在一个充满星光的小世界中探索。

通过寻找散落的“星光碎片”，帮助主人公完成一个特别的约定。

最终章节：

玩家完成所有收集后，将开启最终生日祝福。

结尾显示：

> 生日快乐

---

# 3. 核心体验

玩家体验：

- 探索未知地图
- 发现隐藏故事
- 与NPC交流
- 解开谜题
- 收集回忆
- 完成最终祝福

---

# 4. 游戏定位

目标：

制作一款小型但完整的个人纪念 RPG。

重点不是战斗规模。

重点：

故事、情感、回忆。
